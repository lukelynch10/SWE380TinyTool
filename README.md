# SWE380TinyTool

In order to view and use the application you need have VS code installed with a live server. From there make sure you run the index.html file as this will bring up the main page. From there you should be able to run the program, add and delete components, add and delete methods, enter in data for each component,etc... The processes for adding, deleteing, and editing both components and methods are described in the help section which you can go to by clicking the help button located in the top right corner of the screen. It also contains all information needed to understand how to use the calculator, the formulas used in the calculations, and what your scores say about the level of cohesion and coupling in your program.
